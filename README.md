# 🏥 Hospital Management System (Oracle PL/SQL)  - Clone

This project contains a complete database design for a Hospital Management System.

In my HMS database project, I used core relational database concepts such as designing tables in a relational model with proper primary keys for unique identification and foreign keys to maintain relationships between tables like Patient, Doctor, Visit, and Admission. I applied normalization to reduce data redundancy by separating entities like Gender and Department into different tables and enforced constraints such as primary key, foreign key, and default values to ensure data integrity. I also used Oracle-specific features like sequences and triggers for automatic ID generation, and implemented PL/SQL programming including loops, cursors, and exception handling for business logic. Additionally, I performed CRUD operations, used joins and views for efficient data retrieval, and maintained data consistency through transactions using COMMIT and ROLLBACK.

## Tech Used
- Oracle SQL
- PL/SQL
